Class:
- Blueprint / design
- Defines variables and methods
- Does not occupy memory

Object:
- Real instance of class
- Created using new keyword
- Occupies heap memory

Object Creation Syntax:
ClassName obj = new ClassName();

new keyword:
- Allocates memory
- Calls constructor

Constructor:
- Special method
- Same name as class
- No return type
- Called automatically

Encapsulation:
- Data hiding using private
- Access using getter & setter

Example:
Animal a = new Animal();
