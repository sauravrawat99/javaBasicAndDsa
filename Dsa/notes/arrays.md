Array:
- Data structure
- Stores multiple values of same type
- Uses continuous memory

Why Array?
- Fast access using index
- Easy to store large data

Index:
- Starts from 0
- Last index = length - 1

Declaration:
int[] arr = new int[5];

Initialization:
int[] arr = {1, 2, 3};

Access:
arr[0]  → first element

Traversal:
Use for loop or for-each loop

Update:
arr[index] = newValue;

Array Length:
arr.length

Time Complexity:
Access → O(1)
Traversal → O(n)
Search → O(n)

Important Points:
- Fixed size
- Same data type only
